Part 1 · Customer app

Getting in

1
Welcome
welcome
The first screen — the promise in one line.
Point out
  • Tagline “Where Everything Coordinates” on a dark pill
  • Value proposition and hero
  • “Log in” and “Get started” (sign up)
  • Small “← Back to selection”
Say: Open with the tagline — this is the one-sentence pitch the user reads first.
Get here: Launcher → Customer app
2
Sign up
signup
Create an account with explicit consent up front.
Point out
  • Name, email, password
  • Terms & privacy consent + AI-analysis consent
  • “Continue with Google”
  • Link to log in (phone number intentionally not collected)
Say: Point out consent is captured at sign-up, not buried later.
Get here: Welcome → Get started
3
Verify email
verify
Confirms identity with a 6-digit code.
Point out
  • 6-digit code entry (numeric keypad on mobile)
  • Resend to email — no SMS/phone dependency
Say: Email-only keeps it privacy-friendly and simple.
Get here: Sign up → Continue
4
Log in
login
Returning-user sign in.
Point out
  • Email + password
  • Continue with Google
  • Forgot password?
  • New here? Create account
Say: Standard, fast return path.
Get here: Welcome → Log in
5
Forgot password

Say: Show the recovery path exists.

Say: Completes account recovery.
Get here: Forgot → link
