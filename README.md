# Assemble — Wireframes (Version 1)

High-fidelity, greyscale, responsive **clickable prototype** for **Assemble**, an AI
outfit-matching PWA. Covers both the customer app and the single-login admin panel.

## Open it

Open **`Assemble - Prototype.html`** in any modern browser — no build step or
dependencies. From the launcher choose **Customer app**, **Admin panel**, or **Flow charts**.

- Single, self-contained HTML file.
- Responsive — resize the window to see mobile ↔ desktop layouts.
- Intentionally greyscale; brand colour/typography to be applied on top later.
- Interactions carry state (selections, credits, subscription, sharing) so flows
  behave like the real product.
